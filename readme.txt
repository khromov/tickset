Show Tickset events on your website

== Description ==
This plugin lets you embed and display Tickset events on your website.

You can use the Gutenberg editor or shortcodes to embed events anywhere on your site.

**Usage**

*Basic usage*

Once you have installed and activated the plugin, you will see a prompt at the top of the admin area that instructs
you how to install and configure this plugin.

== Requirements ==
* PHP 5.6 or higher
* WordPress 5.0 or higher

== Translations ==
* Swedish

== Installation ==
1. Upload the `tickset` folder to `/wp-content/plugins/`
2. Activate the plugin (Tickset) through the 'Plugins' menu in WordPress
3. See the prompt at the top of the admin area that instructs you how to install and configure this plugin.
